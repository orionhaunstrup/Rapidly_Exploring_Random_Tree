# Rapidly_Exploring_Random_Tree
This was my final project for CSPB 2077 - Data Structures. The assignment was to choose any data structure of interest to you, that was not covered in the course material, and program up a working version of it! I did some research and found a fascinating little-known data structure called a Rapidly Exploring Random Tree. This project is my custom home-made version of that data structure.

The core idea is that you're trying to teach a computer to solve a "maze" of sorts. You lay down a playing field, place some obstacles, and then establish a start point and an end point... and a Rapidly Exploring Random Tree is a data structure designed to find a path from the start to the finish.

I chose this project to work on because not only is it frankly kind of fun, but it's also very visually satisfying and artistic. I designed my working version of it to be particularly colorful.

Have fun playing with it!
